Для запуска вам потребуется зайти в виртуальную среду следующим образом:
    Запустите MS Powershell, с помощью "cd" откройте текущую директорию
    Выполните команду: 
        Scripts\Activate.ps1
Теперь, необходимо запустить fastAPI
    Выполните команду в виртуальной среде:
        fastapi dev main.py

После этого вы можете запустить в браузере страницы:
    http://127.0.0.1:8000/select/{table}
    Эта команда выводит все записи из таблицы указанно вместо {table}, в нашей базе есть 2 таблицы order и reservation
    
    http://127.0.0.1:8000/delete/{table}/{id}
    Эта команда удалит запись из указанной таблицы по указанному id

    http://127.0.0.1:8000/addOrder/{fio}/{phone}/{price}
    Эта команда добавить новый заказ

    http://127.0.0.1:8000/addReserv/{dateStr}/{order_id}
    Эта команда добавить новую бронь, дату вводить по формату "год_месяц_день_часы_минуты_секунды", пример: "2024_9_16_15_0_0"
    
    http://127.0.0.1:8000/changeStatus/{order_id}/{status}
    Эта команда изменяет статус заказа, статусы бывают следующие: 1 - оплачен, 2 - не оплачен, 3 - отменён
